🎲 Dice Roller Simulator :-

A simple Python program that simulates rolling a six-sided dice. Roll the dice as many times as you like and see what number comes up!

Features :-

Roll a virtual six-sided dice.
Press Enter to roll.
Option to roll again or exit.
Fun and easy to use.

How to Use :-

Make sure you have Python 3 installed.
Download the dice_roller.py file.
Open a terminal or command prompt and navigate to the project folder.

Run the program :-

p1.py
Press Enter to roll the dice.
After each roll, you can choose to roll again or quit.

Example Output :-

🎲 Welcome to the Dice Roller Simulator! 🎲
Press Enter to roll the dice...
You rolled a 4!

Do you want to roll again? (y/n): y
Press Enter to roll the dice...
You rolled a 6!

Do you want to roll again? (y/n): n
Thanks for playing! Goodbye!
